## Key-Bindings:


####Linux:
```
Key: 'ctrl+shift+g' Git add all,commit and push
Key: 'ctrl+shift+.' Git add working file,commit and push
```

####Mac(OSX) and Windows:
```
Key: 'super+shift+g' Git add all,commit and push
Key: 'super+shift+.' Git add working file,commit and push
```
